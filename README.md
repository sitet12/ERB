# Laundry Management Application

This is a simple web application built with **Laravel**, **React**, and **Inertia.js**.

The project is designed for a **laundry business** to manage:
- Client revenues
- Laundry charges (expenses)
- Monthly benefit (profit)
- Comparison between the current month and the previous month
